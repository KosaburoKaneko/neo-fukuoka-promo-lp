# ネオフクオカ 父の日ギフトLP

## 概要

このプロジェクトは、2026年6月21日の父の日に向けて、ネオフクオカの商品を父の日ギフトとして訴求するLPを制作するためのものです。

LPはBASEとは別ドメインで公開し、CTAからネオフクオカ公式オンラインストアの各商品ページへ遷移させます。

## 目的

父の日ギフトを探しているユーザーに対して、ネオフクオカの商品を「お父さんの晩酌やごはん時間に合う、少し気の利いた食品ギフト」として訴求し、公式オンラインストアでの購入につなげることを目的とします。

## 主な訴求内容

* ネオフクオカの商品は父の日ギフトにおすすめ
* 甘いものが苦手なお父さんにも贈りやすい
* 晩酌、おつまみ、ごはんのお供に使いやすい
* ネオ柚子胡椒、激辛ネオ柚子胡椒、ごはんにかけるもつ鍋醤油味の3商品を好みに合わせて選べる
* 高級感のあるギフトボックス付きのギフトセットがある
* 好きな商品を組み合わせられる3本セットがある

## LPの前提

* LPはBASEとは別ドメインで制作する
* CTAリンクはBASEの商品ページへ遷移させる
* CTAリンクにはUTMパラメータを付与する
* 商品URLと商品画像はネオフクオカ公式オンラインストアから取得する
* 足りない画像がある場合は、仮画像で埋めずに不足画像リストとして報告する

## 公式オンラインストア

https://store.neo-fukuoka.jp/

## 主要対象商品

* 〖ギフト用〗選べる3本ギフトセット
* 選べる3本セット
* ネオ柚子胡椒3本セット
* 激辛ネオ柚子胡椒3本セット
* ごはんにかけるもつ鍋醤油味3本セット

## 推奨ディレクトリ構成

Codexは、必要に応じて以下のような構成を作成してください。

```txt
.
├── README.md
├── AGENT.md
├── package.json
├── fathersday_2026
│   ├── index.html
│   ├── styles
│   │   └── main.css
│   ├── scripts
│   │   ├── main.js
│   │   └── tracking.js
│   ├── data
│   │   └── products.json
│   └── assets
│       ├── images
│       └── icons
├── scripts
│   └── fetch-store-assets.js
└── reports
    └── missing-assets.md
```

実装環境に合わせて、Vite、Next.js、Astro、静的HTMLなどの構成に変更して構いません。ただし、静的LPとして公開しやすく、テキストや画像を修正しやすい構成にしてください。

## 実装完了時に確認すること

* スマートフォン表示でファーストビューが自然に見えること
* CTAが複数箇所に設置されていること
* CTAリンクにUTMが付与されていること
* 商品カードからBASEの商品ページへ遷移できること
* 商品画像が取得・配置されていること
* 不足画像がある場合は `reports/missing-assets.md` に記載されていること
* 「刺激」など辛さだけを想起させる表現を使っていないこと
* 健康効果、効能、断定的な景表法リスク表現を使っていないこと
