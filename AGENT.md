## LPの目的

父の日ギフトを探しているユーザーに対して、ネオフクオカの商品を「お父さんの晩酌やごはん時間に合う、少し気の利いた食品ギフト」として訴求し、公式オンラインストアでの購入につなげること。

## ターゲット

メインターゲットは、父の日ギフトを探している30〜50代の男女です。

特に以下のユーザーを想定してください。

* お父さんに食べ物や調味料を贈りたい人
* お酒好きのお父さんに合うギフトを探している人
* 甘いもの以外の父の日ギフトを探している人
* 毎年ビールやお酒ばかりで、今年は少し違うものを贈りたい人
* 高すぎないが、きちんと感のあるギフトを探している人
* 公式LINE、Instagram、Meta広告経由でLPに訪れる既存顧客またはブランド認知済みの人

## LP全体の訴求方針

父の日らしい「感謝」だけに寄せすぎず、食卓で実際に使うシーンを重視してください。

訴求軸は以下です。

* いつもの晩酌を少し楽しくする
* お父さんのごはん時間をちょっと贅沢にする
* 甘いものが苦手なお父さんにも贈りやすい
* お酒のおつまみ、ごはんのお供、ピザ、パスタ、カレー、ネギトロ、チーズなどに使える
* 福岡らしさのある、少し気の利いたギフト
* 高級感のあるギフトボックスで、父の日らしいきちんと感が出せる

避ける方向性は以下です。

* 過度に感動的・泣かせる父の日表現
* 高級すぎる贈答品のような表現
* 健康効果や効能をうたう表現
* 「絶対喜ばれる」など断定的すぎる表現
* 辛さだけに寄せた訴求
* 「刺激」という表現

## UTM設計

LPへの流入元ごとに、CTA遷移時のUTMをBASE商品ページへ引き継いでください。

### Meta広告用

```txt
utm_source=meta
utm_medium=paid_social
utm_campaign=fathersday_2026
utm_content=lp_cta
```

### Instagram投稿用

```txt
utm_source=instagram
utm_medium=social
utm_campaign=fathersday_2026
utm_content=lp_cta
```

### Instagramストーリーズ用

```txt
utm_source=instagram
utm_medium=stories
utm_campaign=fathersday_2026
utm_content=lp_cta
```

### LINE配信用

```txt
utm_source=line
utm_medium=message
utm_campaign=fathersday_2026
utm_content=lp_cta
```

### CTA位置別utm_content

CTA位置別に識別できるよう、可能であれば `utm_content` を以下のように出し分けてください。

* fv_cta
* reason_cta
* giftbox_cta
* product_card_giftset
* product_card_select3
* product_card_neo
* product_card_spicy
* product_card_motsunabe
* bottom_cta

### UTM引き継ぎ仕様

LPアクセス時のURLパラメータをJavaScriptで読み取り、CTA遷移時にBASEの商品URLへ引き継いでください。

LPにUTMが付与されている場合は、その値を優先してください。

LPにUTMが付与されていない場合は、以下のデフォルト値を使用してください。

```txt
utm_source=lp
utm_medium=owned
utm_campaign=fathersday_2026
```

`utm_content` はCTA位置ごとに設定してください。

## 実装上の注意

* スマートフォン表示を最優先してください
* PC表示でも崩れないレスポンシブ対応にしてください
* 商品カードから各BASE商品ページへ遷移できるようにしてください
* 公式オンラインストアから取得した商品URLと画像を使用してください
* URLや画像が取得できない場合は、仮で済ませず `reports/missing-assets.md` に記載してください
* テキストや画像を後から差し替えやすい構造にしてください
* LP上部から購入導線までのスクロール距離を長くしすぎないでください
* 「刺激」という表現は使用しないでください
* 辛さだけに見えるデザインにしないでください
* 未確認の配送、包装、のし、メッセージカード対応は断定しないでください
* 実際の商品ページに記載されている情報と矛盾しないようにしてください
